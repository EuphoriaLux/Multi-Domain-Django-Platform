Implementation Instructions

  • Copy the icons/ folder to your static assets directory

  • Upload manifest.json to enable PWA support
  • Use profile images from profiles/ for social media accounts
  • Upload banners from banners/ to respective platforms
  • Use brand-colors/colors.css for consistent color variables
