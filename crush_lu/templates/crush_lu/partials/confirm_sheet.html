{% load i18n %} {% comment %} App-style confirmation dialog: a native rendered as a bottom sheet on mobile and a centered modal on lg+. confirm-sheet.js opens it for every HTMX hx-confirm (replacing the browser's window.confirm) and exposes window.crushConfirm(message, {style, confirmLabel}) -> Promise for vanilla JS callers. Buttons on the triggering element may customize it via data-confirm-style="neutral" and data-confirm-label="...". showModal() provides the focus trap, Esc handling and focus restore. {% endcomment %}
