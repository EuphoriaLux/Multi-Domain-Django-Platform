{% trans "1. Our Commitment" %}

{% trans "Crush.lu is a verified-identity dating and social platform operating exclusively in Luxembourg. We maintain a zero-tolerance policy toward child sexual abuse and exploitation (CSAE) in any form. These standards apply to all users, content, and interactions on the Crush.lu platform, website, and mobile apps." %}

{% trans "We are committed to implementing and enforcing robust safeguards to prevent our platform from being used for harm against minors, and to cooperating fully with law enforcement and child protection authorities when required." %}

{% trans "2. Platform Age Requirements" %}

{% trans "Crush.lu is exclusively an adult platform. Participation requires:" %}

  • {% trans "Minimum age of 18 years." %}
  • {% trans "Mandatory email address verification at sign-up." %}
  • {% trans "Phone number verification (SMS OTP or WhatsApp OTP) before accessing member features." %}
  • {% trans "Identity verification by a Crush.lu coach at a live event, or self-service government identity verification via LuxID (the Luxembourg national digital identity service), before a profile is approved." %}

{% trans "Age is verified against the self-declared date of birth during profile creation and is additionally attested for users who complete LuxID verification, which links directly to Luxembourg government identity records." %}

{% trans "Profiles that cannot be verified or that present credible evidence of underage status are suspended immediately and not approved." %}

{% trans "3. Prohibited Content and Conduct" %}

{% trans "The following are strictly prohibited on Crush.lu and will result in immediate account suspension and reporting to authorities:" %}

  • {% trans "Any content that sexually depicts, exploits, or solicits minors (Child Sexual Abuse Material — CSAM)." %}
  • {% trans "Grooming behaviour targeting minors or adults." %}
  • {% trans "Solicitation or facilitation of sexual contact with minors." %}
  • {% trans "Sharing, distributing, or linking to CSAM in any form." %}
  • {% trans "Misrepresenting one's age or identity for the purpose of gaining access to minors." %}
  • {% trans "Any attempt to circumvent age verification or identity checks." %}

{% trans "4. Prevention Measures" %}

{% trans "Crush.lu implements multiple layers of prevention:" %}

  • {% trans "Mandatory identity verification:" %} {% trans "Every member must verify their identity through a coach-facilitated in-person review or via LuxID before their profile is activated." %}
  • {% trans "Coach-reviewed profiles:" %} {% trans "Crush.lu coaches screen every submitted profile for authenticity, including reviewing profile photos and stated information, before approving membership." %}
  • {% trans "Photo upload controls:" %} {% trans "All profile photos are validated for format and authenticity and reviewed by a Crush.lu coach prior to publication." %}
  • {% trans "Rate-limited communication:" %} {% trans "Connection and communication features are gated behind verified and approved profile status." %}
  • {% trans "No anonymous browsing:" %} {% trans "All member discovery features require a fully verified and approved profile." %}
  • {% trans "Platform by invitation and coach-facilitation:" %} {% trans "New members are introduced to the platform through coach-facilitated events or direct invitation, adding a human layer of vetting." %}

{% trans "5. Detection and Moderation" %}

{% trans "Crush.lu operates an active moderation programme:" %}

  • {% trans "All profile submissions are reviewed by a Crush.lu coach before activation." %}
  • {% trans "Reported profiles and content are reviewed by the Crush.lu moderation team, typically within 24 hours." %}
  • {% trans "Accounts that receive multiple reports or a single credible report of CSAE-related conduct are immediately suspended pending review." %}
  • {% trans "Coach-mediated connections mean that member interactions occur in a structured, human-supervised context." %}

{% trans "6. Reporting CSAE" %}

{% trans "Any user or third party who encounters suspected child sexual abuse or exploitation on the Crush.lu platform is urged to report it immediately." %}

{% trans "Report to Crush.lu" %}

  • {% trans "Safety email:" %} safety@crush.lu {% trans "(monitored daily; urgent matters handled within 24 hours)." %}
  • {% trans "In-app:" %} {% trans "Use the Report/Block function available on any member profile or connection card within the app or website." %}

{% trans "Report to National Authorities" %}

{% trans "If you believe a child is in immediate danger, contact emergency services (112) immediately. You may also report to:" %}

  • {% trans "Luxembourg Police:" %} police.public.lu — {% trans "Tel: 113" %}
  • {% trans "NCMEC CyberTipline (global):" %} missingkids.org/cybertipline
  • {% trans "IWF (Internet Watch Foundation):" %} iwf.org.uk/report
  • {% trans "INHOPE (global hotline network):" %} inhope.org

{% trans "Important:" %} {% trans "Crush.lu will cooperate fully with law enforcement investigations, including providing relevant data in accordance with applicable law. We do not warn suspected offenders when their account is under investigation by authorities." %}

{% trans "7. Enforcement" %}

{% trans "Upon receiving a credible report of CSAE-related conduct or content, Crush.lu will:" %}

  1. {% trans "Immediately suspend the reported account." %}
  2. {% trans "Preserve all relevant account data and communications." %}
  3. {% trans "Report to the appropriate national authority and, where applicable, submit a CyberTip to NCMEC." %}
  4. {% trans "Permanently ban confirmed offenders and take steps to prevent re-registration." %}
  5. {% trans "Cooperate with any law enforcement investigation, providing data as required by applicable law." %}

{% trans "Accounts suspended under this policy are not entitled to appeal while a law enforcement investigation is active." %}
